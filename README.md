# jimkaj.github.io

Source for my personal website, published at [jimkaj.github.io](https://jimkaj.github.io).

Plain static HTML/CSS/JS styled with [Bootstrap 5](https://getbootstrap.com/) and
[Bootstrap Icons](https://icons.getbootstrap.com/) via CDN — no build step, no framework. Served directly by
GitHub Pages from the `master` branch root.

## Structure

- `index.html` — home page: Bio / Projects / Selected Publications / Resume tabs
- `biography.html` — full biography subpage
- `docs/` — static assets (photo, resume PDF, publication PDFs under `docs/papers/`)

## Previewing locally

Open `index.html` directly in a browser — there's nothing to build or serve.
